This folder contains display drivers:

disphstx ... output to DVI (HDMI) and VGA display via HSTX RP2350 interface

dvi ... output to DVI (HDMI) with frame buffer

minivga ... output to VGA monitor with frame buffer

st7789 ... ST7789 controller with output to TFT display with resolution
	320x240 pixels, 16 bits per pixel in RGB565 format

picovga8 ... PicoVGA driver with 8-bit output to VGA.
	Functional, but not properly - higher video modes are not functional.
